전체 키워드
왼쪽에서 키워드를 클릭하면
응대 매뉴얼이 여기에 표시됩니다.

⚙️ 설정

⚠️ 저장되지 않은 변경사항이 있습니다. 아래 [GitHub에 저장] 버튼을 눌러야 모두에게 반영됩니다.

1. 내 정보 (수정하려면 필수)

이름은 수정 기록에 남습니다. 접근 토큰은 관리자에게 받은 공용 토큰을 입력하세요. 이 정보는 이 컴퓨터에만 저장됩니다.

2. 매뉴얼 항목 관리

항목을 추가·수정·삭제한 뒤 반드시 아래 [GitHub에 저장]을 눌러주세요.

3. GitHub에 저장 (모두에게 반영)

저장하면 팀 전체가 새 내용을 보게 되며, 누가 언제 수정했는지 '수정 기록' 탭에 자동으로 남습니다.

4. 저장소 연결 정보

보통 자동으로 인식되므로 바꿀 필요가 없습니다. 화면 주소가 github.io가 아닐 때만 직접 입력하세요.

✏️
왼쪽 매뉴얼 항목 관리에서 [편집]을 누르거나
[+ 새 항목 추가]를 누르면 여기에 편집 화면이 나타납니다.

매뉴얼 내용

줄 앞에 ■ 를 쓰면 소제목, - 를 쓰면 목록, "..." 로 감싸면 응대 멘트 상자, [이미지:...] 줄은 이미지로 표시됩니다.

또는 이미지를 복사한 뒤 아래 내용 칸에 Ctrl+V로 붙여넣으세요. 커서가 있는 위치에 들어갑니다.
