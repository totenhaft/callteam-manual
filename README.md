# 콜팀 응대 매뉴얼

전화 응대 시 환자 증상을 입력하면 관련 키워드가 감지되고, 클릭 한 번으로 응대 매뉴얼을 확인할 수 있는 웹앱입니다.

## 사용 방법
1. 상단 입력란에 환자가 말한 증상을 입력하면 왼쪽에 관련 키워드가 노란색으로 표시됩니다.
2. 키워드를 클릭하면 오른쪽에 확인 질문·응대 멘트·조치·에스컬레이션 기준이 표시됩니다.

## 매뉴얼 수정
1. **설정** 탭에서 이름과 관리자에게 받은 공용 접근 토큰을 입력 후 저장합니다. (최초 1회)
2. 항목을 추가·수정·삭제한 뒤 **[GitHub에 저장]** 버튼을 누르면 팀 전체에 반영됩니다.
3. **수정 기록** 탭에서 누가 언제 수정했는지 확인할 수 있습니다.

## 파일 구성
- `index.html` — 웹앱 화면 전체
- `data.json` — 매뉴얼 내용 (수정 시 이 파일이 커밋됩니다)

## 관리자: 공용 토큰 발급
GitHub → Settings → Developer settings → Fine-grained personal access tokens에서
이 저장소만, `Contents: Read and write` 권한만 부여한 토큰을 만들어 팀에 공유하세요.
